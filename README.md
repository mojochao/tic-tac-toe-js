# tic-tac-toe-js

This is a stupid self-playing tic-tac-toe game for fun with es6.

## Installation

```
$ git clone https://github.com/mojochao/tic-tac-toe-js.git
$ cd tic-tac-toe-js
$ npm install
$ npm test
$ npm start
```
